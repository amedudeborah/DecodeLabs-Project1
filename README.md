# E-Commerce Data Cleaning Project

## Overview

This project involved cleaning and preparing an e-commerce dataset using Microsoft Excel. The dataset contained order information, customer details, payment methods, and sales records.

## Tasks Performed

* Removed duplicate records
* Checked for missing values
* Corrected data formatting
* Standardized data entries
* Prepared the dataset for analysis

## Tool Used

* Microsoft Excel
